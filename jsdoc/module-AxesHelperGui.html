 AxesHelper folder name. Default is lang.axesHelper
options.cookie Cookie <optional>
 Your custom cookie function for saving and loading of the AxesHelper settings. Default cookie is not saving settings.
options.cookieName string <optional>
 Name of the cookie is "AxesHelper" + options.cookieName. Default is undefined.
options.camera THREE.PerspectiveCamera <optional>
 camera. Set the camera if you want to see text size is independent from camera.fov. The text height will be calculated as textHeight = camera.fov * textHeight / 50 See https://threejs.org/docs/index.html#api/en/cameras/PerspectiveCamera.fov about camera.fov. Default is undefined. Default camera.fov is 50.
