AxesHelper.

An axis object to visualize the 1, 2 or 3 axes.

Quick start

AxesHelper.

The easiest way to use AxesHelper in your code is import AxesHelper from AxesHelper.js file in your JavaScript module. Example

import { AxesHelper } from 'https://raw.githack.com/anhr/AxesHelper/master/AxesHelper.js';

or

import { AxesHelper } from './AxesHelper.js';

Now you can use AxesHelper in your javascript code. See AxesHelper API for details.
